## Python 数据类型

计算机中有海量的数据，为了有效利用计算机资源、有效根据我们的需求进行选择，如何有效的对数据进行分类，如何更好的表达数据本身，就显得非常必要。

#### Variable in Python:

- Numbers

  - int (integer)

  - float (floating-point number)
  - complex

- Bool (true / false)

- String

- Tuple

- List

- Set

- Dictionary

#### Type conversion

仅需将需转换的数据放进待转类型的函数即可

```python
>>> a = float(18)
>>> print(a)
18.0
```
